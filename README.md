# korge-dragonbones

SWF (Adobe Animate) plugin for KorGE using kproject.

## Usage

By using kproject:

### `deps.kproject.yml`

```kotlin
dependencies:
  - git::korge-swf::korlibs/korge-swf::/korge-swf::f7af9d42e39ce7914f7f1e03d7ca404c8f4a50d8
```
